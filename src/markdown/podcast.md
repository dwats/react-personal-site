# Podcast API & Player

This full stack application takes a raw XML RSS feed, converts it into JSON, exposes
endpoints, and then presents parsed JSON content to the user via this React app.

## Tech Stack
* Serverless
* AWS Lambda
* NodeJS 8.10
* React 16.3.2

## Testing
* Mocha
* Chai
* Sinon
* Rewire
* Istanbul

## Continuous Integration
* Travis-CI
* Coveralls
